# designPatterns

git pull origin master (se você não está numa branch)

# HEAD

* Adicionar arquivos:
git add file.txt
git add .

* Remover arquivos:
git rm -r file.txt

# LOCAL REPO

* git commit -m "meu commit"
* git coomit -a (Daí abre o editor de texto e você faz o comentário)

# REMOTE REPO

git push origin master (se você não está numa branch)
